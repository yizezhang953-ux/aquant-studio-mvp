# 风险披露与免责声明

版本：v0.1  
适用阶段：MVP / 沙箱 / 模拟盘  

## 1. 非投资建议

本系统提供的策略编辑、回测、参数优化、模拟盘和报告分析功能，仅用于量化研究、学习和系统验证，不构成任何投资建议、买卖建议、收益承诺或资产管理服务。

## 2. 回测不代表未来表现

历史回测结果基于历史数据、模型假设、手续费、滑点和成交规则模拟生成。历史表现不代表未来收益，任何策略都可能在未来市场环境中失效。

## 3. 模拟盘不等于实盘

模拟盘使用历史行情或模拟行情生成虚拟订单和虚拟成交，不连接真实券商，不发送真实订单，不涉及真实资金。模拟成交结果不保证能在真实市场中复现。

## 4. 用户自行承担风险

如果未来系统接入真实交易，用户应充分理解策略逻辑、交易规则、市场风险、流动性风险、技术故障风险和合规风险，并自行承担由此产生的全部风险。

## 5. 实盘功能需单独审批

当前 MVP 默认禁止实盘交易。任何真实交易功能上线前，必须经过独立的安全评估、风控评估、合规评估、密钥管理评估和人工审批。
