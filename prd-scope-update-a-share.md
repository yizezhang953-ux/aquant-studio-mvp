# PRD 范围修订说明：初版仅支持 A 股

日期：2026-06-20  
关联文档：`quant-model-platform-prd.md`  
修订原因：第一阶段 PRD 中曾将“中国 A 股”和“加密货币现货”作为初版市场二选一。现确认初版应用只设计股票市场，优先支持中国 A 股。

## 1. 修订结论

MVP 初版市场范围确定为：中国 A 股。

初版不支持：

- 加密货币。
- 港股。
- 美股。
- 期货。
- 外汇。
- 真实资金实盘交易。
- Tick 高频交易。
- Level 2 订单簿数据。

## 2. MVP 数据范围

| 项目 | 初版范围 |
|---|---|
| 市场 | 中国 A 股 |
| 标的 | 沪深 A 股单只股票 |
| 股票代码示例 | `600519.SH`、`000001.SZ`、`300750.SZ` |
| 频率 | 日线、60 分钟、30 分钟、15 分钟 |
| 价格类型 | 开盘价、最高价、最低价、收盘价 |
| 成交数据 | 成交量、成交额 |
| 复权方式 | 前复权、后复权、不复权 |
| 回测类型 | 单标的历史回测 |

## 3. 对 PRD 的影响

需要在后续版本 PRD 中统一替换：

- 将“股票或加密货币”改为“A 股股票”。
- 将 `market: crypto_spot` 示例改为 `market: a_share`。
- 将 `symbol: BTCUSDT` 示例改为 `symbol: 600519.SH`。
- 模板策略默认标的与数据频率均以 A 股为准。
- 数据中心只展示 A 股数据覆盖情况。

## 4. 对原型设计的影响

第二阶段原型已经按 A 股边界设计：

- 左侧提示明确写入“仅支持中国 A 股单标的历史回测”。
- 策略编辑器使用 A 股股票代码示例。
- 数据中心显示 A 股支持范围，并标记加密货币不在初版范围。
- 回测报告使用 A 股标的示例 `600519.SH`。

## 5. 后续建议

第三阶段技术方案应优先围绕 A 股设计：

- A 股行情数据表。
- 股票基础信息表。
- 复权价格处理方案。
- 交易日历处理方案。
- 涨跌停、停牌、手续费与滑点规则。
- A 股 T+1 交易限制是否纳入 MVP 的产品决策。
