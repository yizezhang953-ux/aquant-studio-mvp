# 阶段 11 交付物：回测参数差异视图

版本：V2.1 Stage 11  
日期：2026-06-22  
状态：已完成前端参数差异高亮。

## 1. 阶段目标

在回测对比中自动指出不同回测之间的关键参数差异，帮助用户理解“为什么这几次回测结果不一样”。

## 2. 已交付内容

- 回测对比面板新增“参数差异”区域。
- 选择 2 条及以上回测时自动计算差异。
- 仅展示有变化的字段。
- 差异字段高亮显示。
- 支持比较最多 4 条回测。

## 3. 当前比较字段

- 股票
- 周期
- 开始日期
- 结束日期
- 入场方向
- 入场阈值
- 出场方向
- 出场阈值
- 单笔仓位
- 最大仓位
- 止损
- 止盈
- 最大回撤约束

## 4. 使用流程

1. 登录前端。
2. 运行多次不同参数的回测。
3. 在“我的回测记录”中勾选“对比”。
4. 查看“回测对比”下方的“参数差异”。
5. 如果参数一致，系统显示核心参数一致。

## 5. 验收标准

- TypeScript 检查通过。
- Vite 构建通过。
- 后端测试通过。
- 结构检查通过。
- 前端可以高亮显示不同回测的参数差异。
