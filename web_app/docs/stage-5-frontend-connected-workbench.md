# 阶段 5 交付物：前端连接与策略工作台

版本：V2.1 Stage 5  
日期：2026-06-21  
状态：已完成可点击的前端工作台，连接阶段 4 的账户与策略持久化 API。

## 1. 阶段目标

把后端 API 变成用户可以直接操作的网页界面。用户可以在浏览器里注册、登录、复制模板、保存策略、编辑策略 JSON、查看版本历史，并管理自己的策略列表。

## 2. 已交付内容

- 前端主界面：`web_app/frontend/src/App.tsx`
- 前端样式：`web_app/frontend/src/styles.css`
- 连接后端 API：`http://localhost:8000/api/v1`
- 本地 token 保存：`localStorage.aquant_token`
- 可操作页面：
  - 登录/注册
  - 当前用户状态
  - 模板列表
  - 复制模板为个人策略
  - 我的策略列表
  - 策略 JSON 编辑器
  - 策略状态切换
  - 策略保存与删除
  - 策略版本历史

## 3. 前端功能流

1. 用户进入 `http://localhost:5173`。
2. 用户注册或登录。
3. 前端保存后端返回的 Bearer token。
4. 用户从模板库选择策略模板。
5. 点击复制模板，前端调用 `/api/v1/templates/{template_id}` 和 `/api/v1/strategies`。
6. 用户在 JSON 编辑器中修改策略。
7. 点击保存，前端调用 `PUT /api/v1/strategies/{strategy_id}`。
8. 后端保存策略并生成版本记录。
9. 用户可以从右侧版本历史加载旧版本到编辑器。

## 4. 页面结构

| 区域 | 用途 |
|---|---|
| 顶部状态栏 | 显示产品名、市场、模拟环境和实盘禁用状态 |
| 消息条 | 显示成功、失败和操作提示 |
| 登录/注册面板 | 账户入口 |
| 左侧栏 | 用户状态、模板复制、策略列表 |
| 中间编辑区 | 策略名称、状态和 JSON 编辑器 |
| 右侧栏 | 策略版本历史 |

## 5. 本地运行

后端：

```powershell
cd web_app/backend
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

前端：

```powershell
cd web_app/frontend
npm install
npm run dev
```

访问：

```text
http://localhost:5173
```
