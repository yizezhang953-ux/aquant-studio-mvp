# 阶段 7 交付物：回测前端闭环

版本：V2.1 Stage 7  
日期：2026-06-22  
状态：已完成从策略编辑器一键运行回测，并在前端展示结果。

## 1. 阶段目标

把“编辑策略”和“验证策略”连起来。用户在工作台中选择或编辑策略后，可以直接点击“运行回测”，前端调用后端回测 API，并展示核心指标、权益曲线和交易明细。

## 2. 已交付内容

- 前端新增“运行回测”按钮。
- 前端调用：
  - `POST /api/v1/backtests`
  - `GET /api/v1/backtests/{backtest_id}/report`
- 回测结果面板：
  - 总收益
  - 最大回撤
  - 胜率
  - 交易数
  - 最终权益
  - 夏普值
- 简易权益曲线柱状图。
- 交易明细列表。
- 表单模式和 JSON 模式都可以直接运行回测。

## 3. 使用流程

1. 登录前端工作台。
2. 从模板复制一个策略。
3. 在结构化表单或 JSON 模式中调整策略。
4. 点击“运行回测”。
5. 右侧“回测结果”面板显示指标、权益曲线和交易明细。
6. 如需正式保存策略，再点击“保存策略”生成版本记录。

## 4. 验收标准

- 前端 TypeScript 检查通过。
- Vite 构建通过。
- 后端测试通过。
- 结构检查通过。
- 回测接口联调通过。
- 用户可以不离开工作台完成策略编辑与回测查看。
